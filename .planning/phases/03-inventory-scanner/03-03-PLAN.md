    **Export function `matchInventory`**:
    ```typescript
    export async function matchInventory(
      items: InventoryItem[],
      invocations: InvocationRecord[],
      claudeConfigPath?: string,
    ): Promise<ScanResult[]>
    ```

    Implementation:
    1. Call `buildInvocationMaps(invocations)` to get `{ agents, skills, mcpServers }` maps
    2. Call `readClaudeConfig(claudeConfigPath)` to get `config` (for `skillUsage`)
    3. Extract `skillUsage = config.skillUsage ?? {}` (type: `Record<string, { usageCount: number; lastUsedAt: number }>`)
    4. For each `item` in `items`, produce a `ScanResult`:

       **If `item.category === 'agent'`:**
       - Lookup `agents.get(item.name)` -> `summary: InvocationSummary | undefined`
       - If found: `lastUsedMs = new Date(summary.lastTimestamp).getTime()`, `count = summary.count`
       - If not found: `lastUsedMs = null`, `count = 0`
       - `tier = classifyGhost(lastUsedMs)`
       - `lastUsed = lastUsedMs !== null ? new Date(summary.lastTimestamp) : null`

       **If `item.category === 'skill'`:**
       - First try invocation map: `skills.get(item.name)` -- check both directory name and resolved name
       - Call `resolveSkillName(item.path)` to get the registered name
       - Check `skills.get(registeredName)` as well
       - If invocation match found: use that timestamp and count
       - Else check `skillUsage[registeredName]` or `skillUsage[item.name]` -- iterate skillUsage keys looking for a match containing `item.name`
       - If skillUsage match found: `lastUsedMs = skillUsageEntry.lastUsedAt`, `count = skillUsageEntry.usageCount`
       - If neither matched: `lastUsedMs = null`, `count = 0`
       - Classify with `classifyGhost(lastUsedMs)`

       **If `item.category === 'mcp-server'`:**
       - Lookup `mcpServers.get(item.name)` -> `summary`
       - If found: use timestamp and count. If not: null and 0.
       - Classify with `classifyGhost(lastUsedMs)`

       **If `item.category === 'memory'`:**
       - Memory files use `item.mtimeMs` directly (no invocation matching)
       - `lastUsedMs = item.mtimeMs ?? null`
       - `count = 0` (memory files don't have invocation counts)
       - Classify with `classifyGhost(lastUsedMs)`
       - `lastUsed = lastUsedMs !== null ? new Date(lastUsedMs) : null`

    5. Return all `ScanResult[]`
