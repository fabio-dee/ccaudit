<objective>
Terminal rendering layer: all table builders, header formatter, and health score renderer in @ccaudit/terminal.

Purpose: This package becomes the presentation layer that each CLI subcommand calls with typed data. Every render function returns a string (never prints to stdout) so the CLI command controls output, and the renderers are independently testable. The default ghost view uses column-aligned plain text per D-02, while inventory/mcp/trend use cli-table3 bordered tables.

Output: `packages/terminal/src/tables/` with 6 renderer modules + barrel export. Package wired with picocolors and @ccaudit/internal dependencies.
</objective>
