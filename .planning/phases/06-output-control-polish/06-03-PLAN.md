    ```yaml

    ```

    The `--run` flag ensures vitest does not enter watch mode in CI.

    3. **Keep build job depending on test** -- no changes needed to the build job, it already has `needs: [lint, typecheck, test]`.

    Full updated test job:
    ```yaml
    test:
      strategy:
        matrix:
          os: [ubuntu-latest, macos-latest]
      runs-on: ${{ matrix.os }}
      steps:
        - uses: actions/checkout@v4
        - uses: pnpm/action-setup@v4
        - uses: actions/setup-node@v4
          with:
            node-version: 22
            cache: pnpm
        - run: pnpm install --frozen-lockfile
        - run: pnpm test -- --run --coverage --coverage.thresholds.lines=80 --coverage.thresholds.functions=80 --coverage.thresholds.branches=80 --coverage.thresholds.statements=80
    ```

    Leave lint, typecheck, and build jobs unchanged (they only need ubuntu-latest).
