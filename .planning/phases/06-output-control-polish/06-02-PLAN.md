<objective>
Wire shared output flags (--quiet, --csv, --ci) into all four CLI commands, add exit code semantics, standardize JSON meta envelope, add CSV export paths, move verbose to stderr, and add quiet TSV/compact output modes.

Note: --no-color is intentionally NOT included in shared args per D-07. It is detected at root level by initColor() reading process.argv directly, which means `ccaudit --no-color ghost` works regardless of gunshi subcommand parsing. See Plan 01 for the initColor() implementation.

Purpose: This is the core output control wiring that makes ccaudit CI-ready and script-friendly. Every OUTP requirement except OUTP-02 (color control, done in Plan 01) is addressed here through command-level changes.
